Privacy Policy / 개인정보처리방침
Effective Date / 시행일: 2026-04-04

Memora는 사용자의 개인정보를 수집하지 않습니다.

모든 데이터는 사용자의 기기에만 저장됩니다.

1. 개요

Memora(패키지명: com.henry.memora)는 개인 개발자 Henry가 개발한 플래시카드 학습 앱입니다. 본 개인정보처리방침은 앱이 사용자의 정보를 어떻게 처리하는지 설명합니다.

2. 수집하는 개인정보

수집하는 개인정보가 없습니다.

Memora는 서버가 없으며, 어떠한 데이터도 외부로 전송하지 않습니다. 회원가입이나 로그인이 필요하지 않으며, 분석(analytics) 도구나 추적(tracking) 기술을 사용하지 않습니다.

플래시카드 데이터, 이미지, 학습 기록, 앱 설정 등 모든 데이터는 사용자의 기기 내 로컬 데이터베이스(SQLite)에만 저장됩니다.

3. 앱 권한

Memora는 다음 권한을 사용하며, 모든 권한은 앱 기능 제공 목적으로만 사용됩니다.

권한 사용 목적
알림 (POST_NOTIFICATIONS) 학습 카드 푸시 알림 전송
다른 앱 위에 표시 (SYSTEM_ALERT_WINDOW) 잠금화면 카드 오버레이 표시
저장소 (READ/WRITE_EXTERNAL_STORAGE) .memk / .mra 파일 가져오기/내보내기 (Android 9 이하에서만 사용)
부팅 시 실행 (RECEIVE_BOOT_COMPLETED) 기기 재시작 후 잠금화면 서비스 복원
정확한 알람 (SCHEDULE_EXACT_ALARM) 정확한 시간에 푸시 알림 전송

위 권한을 통해 수집되는 데이터는 없으며, 어떠한 정보도 외부 서버로 전송되지 않습니다.

4. 데이터 저장 및 삭제

모든 데이터는 사용자의 기기에만 저장됩니다. 앱을 삭제하면 모든 데이터가 함께 삭제됩니다.

사용자는 앱 내 내보내기 기능을 통해 .mra 파일로 데이터를 백업할 수 있으며, .memk 또는 .mra 파일을 가져올 수도 있습니다. 이 파일 작업은 모두 사용자의 기기 내에서만 이루어집니다.

5. 제3자 제공

Memora는 제3자에게 어떠한 데이터도 제공하지 않습니다. 외부 SDK, 광고, 분석 도구를 사용하지 않습니다.

6. 아동 개인정보

Memora는 사용자의 나이를 확인하지 않으며 개인정보를 수집하지 않으므로, 아동의 개인정보가 수집될 가능성이 없습니다.

7. 방침 변경

본 개인정보처리방침이 변경될 경우, 이 페이지를 통해 공지합니다. 중대한 변경이 있을 경우 앱 내 알림을 통해서도 안내합니다.

1. Overview

Memora (package: com.henry.memora) is a flashcard study app developed by Henry, an independent developer. This privacy policy explains how the app handles user information.

2. Data Collection

We do not collect any personal data.

Memora has no server and does not transmit any data externally. No account or login is required. We do not use any analytics or tracking technologies.

All data, including flashcards, images, study records, and app settings, is stored only in the local database (SQLite) on your device.

3. App Permissions

Memora uses the following permissions, all solely for app functionality.

Permission Purpose
Notifications (POST_NOTIFICATIONS) Send study card push notifications
Display over other apps (SYSTEM_ALERT_WINDOW) Show flashcard overlay on lock screen
Storage (READ/WRITE_EXTERNAL_STORAGE) Import/export .memk / .mra files (Android 9 and below only)
Boot Completed (RECEIVE_BOOT_COMPLETED) Restore lock screen service after device restart
Exact Alarm (SCHEDULE_EXACT_ALARM) Deliver push notifications at precise times

No data is collected through these permissions, and no information is transmitted to any external server.

4. Data Storage and Deletion

All data is stored solely on your device. Uninstalling the app will delete all associated data.

You can back up your data as .mra files using the in-app export feature, and import .memk or .mra files. All file operations are performed entirely on your device.

5. Third-Party Sharing

Memora does not share any data with third parties. We do not use any external SDKs, advertising, or analytics tools.

6. Children's Privacy

Memora does not verify user age and does not collect any personal information, so there is no possibility of collecting children's personal data.

7. Changes to This Policy

If this privacy policy is updated, the changes will be posted on this page. For significant changes, we will also notify users through the app.

8. Contact

If you have any questions about this privacy policy, please contact us.

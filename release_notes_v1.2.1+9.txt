<ko-KR>
- 여러 .memk/.mra 파일을 한 번에 가져올 수 있는 일괄 가져오기 화면을 추가했습니다.
- 폴더와 카드 일괄 삭제 안정성을 개선했습니다. 삭제 직후 앱을 백그라운드로 전환해도 깔끔하게 정리됩니다.
- 잠금화면 이름순 정렬이 기기 언어 기준(한글 가나다, 일본어 등)으로 정확하게 정렬됩니다.
- 잠금화면 랜덤 정렬에서 화면을 켤 때마다 새로운 카드가 표시됩니다. 같은 카드만 반복해서 뜨던 문제를 수정했습니다. (순차·이름순 등 다른 정렬도 화면을 켤 때마다 다음 카드로 넘어갑니다.)
- 카드 편집 시 텍스트 입력칸이 내용 길이에 맞춰 자동으로 조정됩니다.
</ko-KR>
<en-US>
- Added a bulk import screen for selecting multiple .memk/.mra files at once.
- Improved batch delete reliability so folders and cards are consistently removed even if the app is sent to the background mid-action.
- Lock screen name sorting now respects your device language (Korean, Japanese, etc.).
- Lock screen random order now shows a fresh card every time you wake the screen, fixing an issue where the same card kept appearing. (Other orders — sequential, by name, and so on — also advance to the next card on each wake.)
- Card edit text fields now resize to fit content length automatically.
</en-US>
